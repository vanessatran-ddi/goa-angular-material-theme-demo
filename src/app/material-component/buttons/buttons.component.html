Buttons Angular Material buttons are native button elements enhanced with Material Design styling and ink ripples. Official Doc

There are three main button types: Primary, Secondary, and Tertiary. If there is only one button on a page, it should be a primary button. There should only ever be one primary button on a page.

GoA Primary Button - Angular Material Raised Buttons
<button mat-raised-button color="primary">Primary</button>
<button mat-raised-button color="primary"><strong>Get started</strong><mat-icon>arrow_right_alt</mat-icon></button>

GoA Secondary Button - Angular Material Stroked Buttons
<button mat-stroked-button color="primary">Secondary</button>
<button mat-stroked-button color="primary" disabled>Secondary</button>

GoA Tertiary Button - Angular Material Basic (Link) Buttons
<button mat-button color="primary">Tertiary</button>

GoA Destructive Button - Angular Material Raised Buttons
<button mat-raised-button color="warn">Destructive primary</button>
